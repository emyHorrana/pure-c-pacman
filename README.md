# Pac-Man in C

This project is a remake of the classic **Pac-Man** game developed entirely in the **C programming language**, without the use of any game engine or external graphics library.

The objective of the project was to explore fundamental programming concepts by implementing all game mechanics from scratch, including map rendering, player movement, collision detection, score tracking, and keyboard input handling.

The game runs directly in the Windows console using the Windows API to control cursor positioning, colors, and screen rendering.

## Features

- Console-based Pac-Man gameplay
- Keyboard-controlled movement
- Collision detection
- Score system
- Custom map rendering
- Colored console interface using the Windows API

## Technologies

- C

## Challenge

This project was developed as an academic programming challenge in which the use of game engines or external graphics libraries was **not allowed**. Every aspect of the game, including rendering and game logic, was implemented manually using only the C language and the Windows console.
